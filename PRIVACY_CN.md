# 隐私政策

## 1. 本地存储机制：密钥零云端

- **存储位置**：API 密钥仅使用浏览器内置的存储机制 (`chrome.storage.local`) 存储在您的本地设备上。它们 **绝不** 会被上传到任何“Nanobrowser”服务器或第三方服务。
- **隔离性**：利用浏览器扩展沙盒，密钥存储与扩展的运行时环境隔离。只有经过授权的扩展代码才能访问这些密钥，防止外部脚本或恶意程序未经授权的访问。

## 2. 用户自主管理

- **可见性控制**：您完全控制密钥的可见性。界面提供切换按钮来显示或隐藏敏感信息（API 密钥、密码），以防止视觉窥探。
- **责任**：虽然我们确保存储安全，但您有责任遵循密钥管理的最佳实践，例如定期轮换和最小权限配置。我们不管理您密钥的生命周期。

## 3. 编码规范：防止泄露

- **提交保护**：敏感配置文件已从版本控制中排除。
- **禁止日志记录**：我们的编码规范严格禁止将敏感密钥记录到控制台或将其包含在错误堆栈中。

## 4. 主动安全机制

- **内容净化**：该扩展包含主动防护措施，以检测并防止在聊天消息或提示词负载中意外包含 API 密钥。
- **提示词安全**：系统提示词旨在忽略提取或操纵敏感密钥信息的尝试。
